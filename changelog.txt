# Changelog 

### Added
- DGIAPI_r.R: DIdb query script
- frag_to_expr.R: FragPipe to intensity matrix (unused)
- LFQpep_to_TMTpep.R: Transform LFQ to TMT format
- quant_matrix_to_frag.R: Intensity matrix to FragPipe LFQ format
- script_protter.R: Protter query script
- transform_spectro.R: Transform spectronaut to FragPipe LFQ format

### Fixed
- customized.R[350]: grep("_diff$", colnames(row_data)) Added a "$" to prevent error when a condition has "diff" in the name
- server.R[1134]: isolated variables to prevent error when changing databases in lines 1864, 1865 & 1884
- server.R[1285]: If no missing values, missval_input() is an empty plot, originally it caused the report downloading to fail.
- server.R[2178]: all reports were downloaded as LFQ_report.rmd
- server.R[1146]: When no differential expressed genes available Heatmap was empty, fixed bug. 

### Modified
- functions.R[get_results_proteins()]: added PEP_LFQ argument to obtain a result table with the column ModPeptides (used in LFQ-peptide pipeline, and future spectro-peptide pipeline).
- functions.R[plot_tsne()]: creates tsne plot
